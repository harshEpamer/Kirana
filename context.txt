## Purpose
A full-stack microservices application for managing a local kirana (grocery) store.
Covers product catalog, inventory management, sales processing, coupon handling, low-stock alerts,
and customer purchase history. Built as a 2-hour hackathon sprint with one agent per microservice.

## Tech Stack
- Language(s): Python 3.11+ (backend), JavaScript/JSX (frontend)
- Framework(s): FastAPI (backend), React 18 via Vite (frontend)
- Database: SQLite3 via SQLAlchemy (single shared file: kirana.db at project root)
- Auth: JWT — python-jose + passlib[bcrypt]; token stored in localStorage as `kirana_token`
- Styling: Tailwind CSS
- Routing: React Router v6
- HTTP Client: Native fetch API (no axios)

## Key Modules / Features
1. Authentication — register with phone+password, login returns JWT, token verified by all services
2. Catalog — browse products by category, search by name, product detail view
3. Inventory — admin CRUD for products, bulk stock adjustments (add/set/sale_deduct)
4. Sales — record a sale with items, apply coupon discount, auto-deduct stock
5. Orders — checkout flow: validate cart + coupon → create sale → return receipt
6. Coupons — two types: order_wise (% or flat off total), product_wise (discount on specific product)
7. Alerts — detect products where stock_qty < reorder_threshold, surface to admin dashboard
8. Customers — list customers, view purchase history per customer

## Database
- Single SQLite file: `kirana.db` at project root
- All services connect via `DATABASE_URL = "sqlite:///../../kirana.db"` (relative from service folder)
- Tables: users, products, sales, sale_items, purchase_history, coupons, stock_adjustments
- Schema + seed data: db/schema.sql
- Initialize: `python scripts/init-db.py`

## Non-Functional Requirements
- CORS: all services allow origin http://localhost:5173
- All responses: JSON with correct HTTP codes (200, 201, 400, 401, 404)
- Auth: JWT HS256, 24-hour expiry, SECRET_KEY env var (default: kirana-secret-key-change-in-production)
- Follows SOLID, KISS, DRY, YAGNI
- No Docker required for hackathon; all services run locally

## Conventions
### Backend
- Entry point: main.py, `app = FastAPI(title="<service-name>")`
- Run: `uvicorn main:app --reload --port <PORT>` (from within service directory)
- DB path from service: `../../kirana.db`
- All routers use prefix e.g. `/auth`, `/inventory`, `/catalog`
- Health check at `GET /health` on every service
- requirements.txt in each service folder: fastapi uvicorn[standard] sqlalchemy python-jose[cryptography] passlib[bcrypt] python-multipart

### Frontend
- JWT token: localStorage key `kirana_token`
- Authenticated requests: header `Authorization: Bearer <token>`
- All API base URLs in `src/config.js`
- Components: functional + hooks only (no class components)
- Forms: controlled via useState
